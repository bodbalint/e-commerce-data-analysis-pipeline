## E-commerce Data Analysis Pipeline (Python + PostgreSQL + Power BI)
This project is a complete end-to-end data analysis pipeline that works with a Brazilian e-commerce database.

## Project Goal
A real-world business data analysis project in which I first clean the data, then analyze it using SQL, and finally visualize it via a Power BI dashboard

## Data Processing Workflow
CSV data -> Python (data loading and preparation) -> PostgreSQL (creating a relational database) -> SQL (writing queries) -> Power BI (dashboard and data visualization)

## Analyses
- Most popular product categories
- Customer satisfaction
- Delivery performance
- Customer analysis
- Revenue trends over time

## Dashboard metrics
- Total revenue
- Total number of orders
- Average rating
- Monthly revenue change
- Revenue by payment type
- Top 10 product categories by number of orders
- Top 5 cities by total number of orders
