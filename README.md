## E-commerce adatelemzés pipeline (Python + PostgreSQL + Power BI)
A projekt egy teljes end-to-end adatelemző pipeline, amely egy brazil E-commerce adatbázisából dolgozik.

## Projekt célja
Egy valós üzleti adatelemzés, ahol először tisztítom, majd elemzem SQL-ben és egy Power BI dashboard-on keresztül vizualizálom az adatokat

## Adatfeldolgozás folyamata
CSV adatok -> Python (adatbetöltés és előkészítés) -> PostgreSQL (relációs adatbázis létrhozása) -> SQL (lekérdezések megírása) -> Power BI (dashboard és adatvizualizáció)

## Elemzések
- Legnépszerűbb termékkategóriák
- Vásárlói elégedettség
- Szállítási teljesítmény
- Vásárlók elemzése
- Bevételi trendek időben

## Dashboard mérések
- Teljes bevétel
- Összes rendelés száma
- Átlagos értékelés
- Havi bevétel változása
- Fizetés típus szerinti bevétel
- Top 10 termékkategória rendelések száma alapján
- Top 5 város összes rendelés szerint
